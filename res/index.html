SeikiGSTool使用指南

SeikiGSTool是一个用于启动GriefSydrome忧郁综合征游戏的启动器，包括修改MOD、改键位、快速联机等功能。

快速开始

单击下列链接以跳转到对应教程。

  • 原版单机启动
  • MOD单机启动
  • 原版联机启动
  • MOD联机启动

原版单机启动

此教程指示您如何使用该工具进行原版单机启动游戏。

  1. 单击“玩原生”单选钮。
  2. 单击“确定”按钮。
  3. 单击“单机”单选钮。
  4. 单击“开始游戏”按钮。

MOD单机启动

此教程指示您如何使用该工具进行MOD单机启动游戏。

  1. 单击“玩MOD”单选钮。
  2. 在“MOD选择与预览”列表中单击选择您想使用的MOD。
  3. 单击“确定”按钮。
  4. 单击“单机”单选钮。
  5. 单击“开始游戏”按钮。

原版多人联机启动

此教程指示您如何使用该工具进行原版多人启动游戏。

  1. 单击“玩原生”单选钮。
  2. 单击“联机”单选钮。
  3. 如果您希望当主机（房主），请单击“一键开房”并在弹出的窗口中选择“标准模式”。当客机请跳过。
  4. 单击“开始游戏”按钮。

MOD多人联机启动

此教程指示您如何使用该工具进行MOD多人联机启动游戏。

  1. 单击“玩MOD”单选钮。
  2. 在“MOD选择与预览”列表中单击选择您想使用的MOD。请在联机前商议好MOD并确保您的MOD与其他联机玩家的MOD一致。
  3. 单击“确定”按钮。
  4. 单击“联机”单选钮。
  5. 如果您希望当主机（房主），请单击“一键开房”并在弹出的窗口中选择“标准模式”。当客机请跳过。
  6. 单击“开始游戏”按钮。
